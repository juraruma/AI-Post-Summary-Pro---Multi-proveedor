Sistema profesional multi-proveedor (OpenAI, Claude, Gemini) para generar resúmenes de posts.

== Description ==

AI Post Summary Pro permite a los editores de contenido ofrecer resúmenes rápidos generados por IA. Con soporte para OpenAI, Claude y Gemini, puedes personalizar el tono y la longitud de cada resumen.

== Installation ==

1. Sube la carpeta del plugin al directorio `/wp-content/plugins/`.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Configura tus API Keys en Ajustes > AI Summary Pro.
