# I-Post-Summary-Pro---Multi-proveedor
Plugin avanzado para WordPress que genera resúmenes automáticos usando OpenAI, Claude y Gemini. Ofrece prompts independientes por IA, control de longitud y un diseño de tarjetas 100% personalizable con Google Fonts. Incluye caché de datos para optimizar el consumo de tokens y un sistema de gestión desde el editor de entradas.
