# AI Post Summary Pro — Multi-Provider

Advanced WordPress plugin that automatically generates post summaries 
using OpenAI, Claude, and Gemini — with independent prompts per 
provider, token optimization, and full visual customization.

## Features

- ✅ Multi-provider support: OpenAI, Anthropic Claude, and Google Gemini
- ✅ Independent prompt configuration per AI provider
- ✅ Summary length control
- ✅ Built-in data cache to minimize token consumption
- ✅ 100% customizable card design with Google Fonts integration
- ✅ Summary management directly from the post editor
- ✅ Uses your own API keys — no subscription required

## Requirements

- WordPress 5.8+
- PHP 7.4+
- An active API key from at least one provider:
  - [OpenAI](https://platform.openai.com/)
  - [Anthropic Claude](https://console.anthropic.com/)
  - [Google Gemini](https://aistudio.google.com/)

## Installation

1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Activate the plugin from the WordPress admin panel
4. Go to **Settings → AI Post Summary Pro**
5. Enter your API key(s) and configure your prompts
6. Open any post and generate your first summary

## Configuration

Each AI provider has its own settings section:
- **API Key** — your personal key from each provider
- **Model selection** — choose the model that fits your needs
- **Custom prompt** — define exactly how you want summaries generated
- **Summary length** — control output size per provider

## Use Case

Ideal for content-heavy WordPress sites, blogs, and news portals 
that want to offer readers a quick AI-generated summary at the top 
of each post — with full control over tone, length, and style.
